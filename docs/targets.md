## Makefile Targets
```
Available targets:

  end-hotfix                          Finish the hotfix and merge it into develop and master, then tag the release
  help                                Help screen
  help/all                            Display help for all targets
  help/short                          This help short screen
  start-hotfix                        Build the current branch as hotfix and merge it into develop and master
  tag                                 Tag the current version only upto minor
  tagall                              Tag the current version upto major and minor
  tagbranch                           Tag the current support branch (no master checkout

```
